# 🧘‍♂️ Sitting-Tracker

A "kawaii" game-themed sitting session tracker designed to help you maintain healthy habits while working, reading, or gaming.

![Vite](https://img.shields.io/badge/Vite-6.0-646cff?logo=vite)

## ✨ Features

*   **Activity Tracking:** Choose from activities like Work, Read, Game, or Relax.
*   **Smart Timer:** Real-time countdown for your sitting sessions.
*   **"Stand Up" Alerts:** Automatic notifications at the 25-minute mark to remind you to stretch and move.
*   **Session History:** Track your progress with a history sidebar, persisted locally in your browser.
*   **Cute Buddy:** An animated character companion that reacts to your activity.
*   **Bouncy UI:** Engaging animations powered by Motion for a tactile, game-like feel.

## 🎨 Aesthetic & Design

The project features a unique "sticker" look with:
*   **Pastel Palette:** Soft pinks, yellows, and lavenders.
*   **Thick Outlines:** Bold borders and custom shadow offsets for a 3D effect.
*   **Modern Typography:** Clean, readable, and fun.

## 🛠️ Tech Stack

*   **Frontend:** React 19 (TypeScript)
*   **Styling:** Tailwind CSS 4
*   **Animations:** Motion (formerly Framer Motion)
*   **Icons:** Lucide React
*   **Bundler:** Vite

## 🚀 Getting Started

### Prerequisites

*   Node.js (Latest LTS recommended)
*   npm

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/situp-cute-tracker.git
    cd situp-cute-tracker
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The app will be available at `http://localhost:3000`.
